INFORMATION TYPE OBJECT

Object Class: Ship report

Acronym: SHPREP

Camel case: ShipReport


Simple Attributes: CATREP; IMOREP;

Complex Attributes: fixedDateRange; featureName; PeriodicDateRange; SourceIndication; TXTCON; NTCTIM;

Associated Information Object: APPLIC;


Definition:

Description of how a ship should report to a maritime authority, including when to report, what to report and whether the format conforms to the IMO standard.


References:

IMO Resolution A 851(20) adopted 27 November 1997


Remarks:

TXTCON is used to describe non-standard ship reports. The Associated Information Object APPLIC indicates characteristics of vessels which use this report.

Distinctions:


Justification:

New information feature

Comment:

No comments.
