Marine Physical Environmental

The following sources are available:

The Marine Physical Environmental Conditions Description as provided to HSSC. Please note that althouh the number S-130 is provided in the document, this number has not been confirmed yet. It is just a number, not more.

The Marine Physical Environmental test data set drafted by Richard (UKHO) some years ago. This file contains also first mapping ideas. But these ideas were very simple and it is assumed that the new data model will be comprehensive.


An Excel spreadsheet based on the scope discussion SNPWG had back in the year 2005 Marine Physical Environmental Conditions Scope wih extensions. This information is extended with a collection of keywords compiled by Wilfred and Briana some years ago.


A clean Excel spreadsheet where your contrubutions and comments will be collected. It is intended to keep this spreadsheet up-to-date to ensure that always the current status is available. The first entries have been made in this spreadsheet as an examble. Marine Physical Environmental Conditions Clean


Marine Physical Environment Background Material

  • Individual contributions to the clean table are available here: Marine Physical Environment Contributions. (Information received as of 22 July, should all have been merged into the main clean table, let me know if anything is missing).
  • updated merged Envrmt Table Combined Contribs.xlsx with comments from group (during work on S-131 Marine Harbour Infrastructure March 2021) and collated further needs from various organizations into word document S-126 Marine Physical Environment


The status of S-126 as of 9/15/21 can be found in the document S-126-PhysicalEnvironment.docx - highlighted text indicates it has been modeled in a UML diagram.
